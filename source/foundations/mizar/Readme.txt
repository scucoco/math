This folder gives the encodings of Mizar and its semeantics in terms of ZFC.

mizar.elf      declares the first order logic and the type system of Mizar
tarski.elf     defines Tarski-Grothendieck set theory as a theory of Mizar

zfc.elf        imports typed set theory from the ZFC folder and defines some additional notions needed for the translation
mizar-zfc.elf  gives the views from the signatures encoding Mizar to those encoding ZFC
zfc-mizar.elf  gives an unfinished view in the opposite direction
